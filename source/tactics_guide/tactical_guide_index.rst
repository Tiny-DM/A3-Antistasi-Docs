Tactical Guide
=============================

.. toctree::
   :maxdepth: 2

.. card::

   The goal of this part of the guide is to create and maintain an easy and fast to use resource-pool for guides and tutorials
   regarding ARMA3, the armaments and mods like ACE3, RHS as well as other useful material. If you wish to add something,
   simply ping the dev-team on discord or open a PR on github and we will find a way to include it in the collection.
